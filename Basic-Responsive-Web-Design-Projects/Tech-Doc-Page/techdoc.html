Introduction

The HTML <head> element contains metadata about the HTML document.

This metadata is not displayed on the webpage, but instead provides the browser with various types of background information.

Metadata usually consists of at least five main elements:

  • document title
  • character set
  • styles
  • links
  • scripts

The following information provides brief explanations of some of the more common elements used within the HTML <head> element.

The <title> Element

The <title> element specifies what the document is titled and is a requirement for all HTML docs.

The contents of the <title> element do the following:

  • creates a title in the tab of the browser
  • provides a title for the page when it is added to bookmarks
  • provides a title for search engine results
The <meta> Element

The <meta> element describes the character set used in the document, keywords, author, page description, and other metadata.

Examples of these main uses of the <meta> element are below:

To define the files character set:

<meta charset="UTF-8">

To create keywords for search engines:

<meta name="keywords" content="HTML, meta, CSS, Javascript">

To designate the author of the page"

<meta name="author" content="John Doe">

To provide a description of the page:

<meta name="description" content="HTML meta element explanation">
The <style> Element

The <style> element is used to create CSS styling within the HTML file itself.

This prevents the need to link to a separate, standalone CSS file, but it also only defines the styling for the individual HTML document in which it exists.

Here is an example of the <style> element:

<style>
body {font-family: helvetica;}
p {color: blue;}
h1 {font-size: 20px;}
</style>
The <link> Element

The <link> element is used to connect a CSS file to the HTML file for styling the webpage.

This can be helpful when you want multiple webpages to have the same overall style.

To achieve this, instead of using the <style> element to input the styling in every HTML file, you can simply create one standalone CSS file and link it to each HTML file you want to style.

Here is an example of how the <link> element is formatted:

<link rel="stylesheet" href="style.css">
The <script> Element

The <script> element is used to provide JavaScript functionality to the webpage.

This can be done by writing the JavaScript directly within the <script> element or, more commonly, by linking to an external JavaScript file.

Here is an example of how to link to an external JavaScript file using the <script> element:

<script src="myScript.js"></script>
